Available Tools

Four MCP tools that give Claude real-time access to customer context

lookup_customer
Retrieves complete customer profile including tier, contract value, API usage, account flags, known issues, and support history.
Returns: CustomerProfile
get_open_tickets
Fetches all open/pending tickets for a customer with age, priority, assignee, and tags for context.
Returns: Ticket[]
get_response_guidance
Generates personalized response templates with tone guidance, escalation paths, and pre-filled customer context.
Returns: ResponseGuidance
check_escalation_rules
Returns SLA requirements by tier including response time targets and auto-escalation triggers.
Returns: EscalationRule

Under The Hood

Built with TypeScript
and the MCP SDK

The server uses Anthropic's official TypeScript SDK to define tools that Claude can call. Each tool connects to a mock database that simulates real enterprise data.

src/index.ts TypeScript 
/**
 * Tool: lookup_customer
 * Retrieves customer account information for personalized support
 */
server.tool(
  "lookup_customer",
  {
    query: z.string().describe("Company name or customer ID")
  },
  async ({ query }) => {
    const customer = database.customers.find(
      c => c.company.toLowerCase().includes(query.toLowerCase())
    );

    // Build personalized context with support guidance
    const context = {
      customer: {
        tier: customer.tier,
        contractValue: `$${customer.contractValue.toLocaleString()}`,
        apiUsage: customer.apiUsage,
        flags: customer.flags,
        knownIssues: customer.knownIssues
      },
      supportGuidance: generateSupportGuidance(customer)
    };

    return { content: [{ type: "text", text: JSON.stringify(context) }] };
  }
);
